# 🚀 Predictive Maintenance using Machine Learning

## 📌 Project Overview
This project focuses on predicting machine failures using Machine Learning to support predictive maintenance in industrial environments. The solution is developed using **IBM watsonx.ai Studio** with **AutoAI**, which automatically builds, trains, and evaluates multiple Machine Learning models to identify the best-performing pipeline.

The model analyzes machine operational parameters and predicts whether a machine is likely to fail or continue operating normally. This helps industries reduce unexpected downtime, improve maintenance planning, and increase operational efficiency.

---

## 🎯 Project Objectives
- Predict machine failures before they occur.
- Reduce unexpected machine downtime.
- Improve equipment reliability.
- Support data-driven maintenance decisions.
- Enhance industrial productivity using AI.

## 📊 Dataset
**Machine Predictive Maintenance Classification Dataset**

Dataset Features:
- Air Temperature
- Process Temperature
- Rotational Speed
- Torque
- Tool Wear
- Product Type
- Machine Failure (Target)
